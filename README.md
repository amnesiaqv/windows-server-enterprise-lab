# Enterprise Active Directory Lab

A production-inspired Windows Server 2025 home lab built in Oracle VirtualBox.

## Overview

This project demonstrates the deployment and administration of a Windows Server enterprise environment including:

* Active Directory Domain Services (AD DS)
* DNS Server
* DHCP Server
* File Server
* Group Policy
* Windows 11 domain clients

## Technologies

* Windows Server 2025
* Windows 11 Pro
* Oracle VirtualBox
* Active Directory
* DNS
* DHCP
* Group Policy
* NTFS Permissions
* PowerShell
