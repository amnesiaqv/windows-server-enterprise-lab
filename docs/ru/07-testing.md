# ✅ Тестирование и проверка инфраструктуры

## Обзор

В этом документе приведены результаты проверки инфраструктуры после развертывания лабораторного проекта **Enterprise Active Directory Lab**.

Цель тестирования заключалась в том, чтобы убедиться в корректной работе всех настроенных служб и подтвердить успешное взаимодействие клиентских компьютеров под управлением Windows 11 с инфраструктурой Windows Server.

## Проверка сетевого подключения

Связь между контроллером домена и обоими клиентскими компьютерами была проверена с помощью протокола ICMP (команда **ping**).

Команда:

```cmd
ping DC01
```

Ожидаемый результат:

* успешные ответы на запросы;
* отсутствие потери пакетов;
* стабильная связь между клиентами и сервером.

---

## Проверка DNS

Работоспособность службы DNS была проверена с помощью следующей команды.

```cmd
nslookup DC01
```

Имя контроллера домена было успешно разрешено, что подтвердило корректную работу DNS-сервера, интегрированного с Active Directory.

---

## Проверка DHCP

Клиентские компьютеры автоматически получили сетевую конфигурацию от DHCP-сервера.

Были проверены следующие параметры:

* IPv4-адрес;
* маска подсети;
* шлюз по умолчанию (если настроен);
* предпочитаемый DNS-сервер;
* DNS-суффикс.

Команда:

```cmd
ipconfig /all
```

---

## Проверка Group Policy

Работа групповых политик была проверена после входа пользователя с доменной учетной записью.

Команда:

```cmd
gpresult /r
```

Результат выполнения подтвердил, что все настроенные объекты групповой политики (GPO) были успешно применены.

---

## Проверка файлового сервера

Общая папка была успешно открыта через автоматически подключенный сетевой диск.

Проверенные возможности:

* автоматическое подключение сетевого диска;
* доступ на чтение;
* доступ на запись (при наличии соответствующих разрешений);
* сетевое подключение к общей папке.

Пример сетевого пути:

```text
\\DC01\Public
```

---

## Проверка Active Directory

Были успешно подтверждены следующие элементы:

* пользователи домена проходят аутентификацию;
* компьютеры Windows 11 успешно присоединены к домену;
* учетные записи компьютеров отображаются в Active Directory;
* организационные подразделения (OU) функционируют корректно;
* группы безопасности доступны для назначения разрешений.

## Заключение

Лабораторный проект **Enterprise Active Directory Lab** был успешно развернут и полностью протестирован.

Все ключевые службы инфраструктуры, включая **Active Directory Domain Services**, **DNS**, **DHCP**, **File Services** и **Group Policy**, корректно функционировали на протяжении всего процесса тестирования.

Созданная среда демонстрирует практические навыки администрирования Windows Server, централизованного управления идентификацией, настройки сетевых служб, развертывания групповых политик и организации корпоративного файлового обмена.

Данный проект максимально приближен к процессам развертывания, администрирования и проверки, которые выполняются в корпоративных инфраструктурах Windows малого и среднего масштаба.
