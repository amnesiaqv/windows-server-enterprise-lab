# ✅ Тестування та перевірка інфраструктури

## Огляд

Цей документ описує процес перевірки, виконаний після розгортання лабораторії Enterprise Active Directory.

Метою тестування було підтвердити, що всі налаштовані служби працюють коректно, а клієнтські комп'ютери Windows 11 можуть успішно взаємодіяти з інфраструктурою Windows Server.

## Мережеве підключення

З'єднання між контролером домену та обома клієнтськими робочими станціями було перевірено за допомогою ICMP (ping).

Команда:

```cmd
ping DC01
```

Очікуваний результат:

* Успішні відповіді
* Відсутність втрати пакетів
* Стабільний зв'язок між клієнтами та сервером

---

## Перевірка DNS

Функціональність DNS була перевірена за допомогою наступної команди.

```cmd
nslookup DC01
```

Ім'я контролера домену було успішно розпізнано, що підтверджує правильну роботу DNS-сервера, інтегрованого з Active Directory.

---

## Перевірка DHCP

Клієнтські комп'ютери автоматично отримали мережеву конфігурацію від DHCP-сервера.

Перевірена інформація включала:

* IPv4-адресу
* Маску підмережі
* Шлюз за замовчуванням (якщо налаштований)
* Бажаний DNS-сервер
* DNS-суфікс

Команда:

```cmd
ipconfig /all
```

---

## Перевірка Group Policy

Розгортання групових політик було перевірено після входу в систему за допомогою доменного облікового запису.

Команда:

```cmd
gpresult /r
```

Результат виконання команди підтвердив, що налаштовані об'єкти групової політики були успішно застосовані.

---

## Перевірка файлового сервера

Спільна папка була успішно доступна за допомогою підключеного мережевого диска.

Перевірена функціональність:

* Автоматичне підключення мережевого диска
* Доступ для читання
* Доступ для запису (де дозволено)
* Мережеве підключення до спільної папки

Приклад мережевого шляху:

```text
\\DC01\Public
```

---

## Перевірка Active Directory

Було успішно перевірено наступні елементи:

* Користувачі домену успішно пройшли автентифікацію
* Клієнти Windows 11 були приєднані до домену
* Облікові записи комп'ютерів з'явилися в Active Directory
* Організаційні підрозділи (OU) працюють коректно
* Групи безпеки доступні для призначення дозволів

## Висновок

Лабораторія Enterprise Active Directory була успішно розгорнута та перевірена.

Усі основні інфраструктурні служби, включаючи Active Directory Domain Services, DNS, DHCP, файлові служби та Group Policy, працювали належним чином протягом усього процесу тестування.

Середовище демонструє практичний досвід адміністрування Windows Server, централізованого керування ідентифікацією користувачів, мережевими службами, розгортанням Group Policy та корпоративним спільним доступом до файлів.

Цей проєкт відображає процеси розгортання, адміністрування та перевірки, які зазвичай виконуються в малих і середніх корпоративних середовищах Windows.
